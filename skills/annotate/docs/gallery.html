Example — comparison table (free HTML in a markdown block)

Two- or three-column weigh-up of options across a fixed set of axes. The columns and cells carry data-annotate-id so users can click a column header or a specific cell to comment on just that piece — the click handler resolves to the closest data-annotate-id ancestor.
